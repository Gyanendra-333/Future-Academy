Course overview

Javascript is a flexible multi-paradigm programming language largely used in the web-development space for both front-end and back-end applications. Whereas HTML and CSS describe the elements on a webpage, code written in JavaScript makes them interactive. A framework such as NodeJS allows back-end code to be written in JavaScript. The tutorials below provide tips and tricks on how to get started with Javascript. JavaScript, often abbreviated as JS, is a high-level, interpreted scripting language that conforms to the ECMAScript specification. JavaScript is a text-based programming language used both on the client-side and server-side that provides interactivity to web pages. JavaScript can be used to create dynamic and interactive websites, and it is also used for game development, server-side programming, and more.
